# Agent Memory & Task Manager

A Python project inspired by AI agent architectures.

## Features

- Create tasks
- Update task status
- Agent state tracking
- JSON persistence
- Task lifecycle management

## Concepts Used

- OOP
- Dataclasses
- Enums
- Type Hints
- Decorators
- Generators
- File Handling
- JSON
- Pathlib
- Custom Exceptions

## Run

```bash
python main.py
```

## Sample Output

Shows task creation, agent state tracking, pending task generation, and JSON persistence.
